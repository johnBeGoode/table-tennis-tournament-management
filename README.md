# Table tennis tournament management
Organize a friendly table tennis tournament within your club or enterprise.

## Steps
1. Create players and pools
2. Put players in pools
3. Enter the scores
4. Enjoy


> [!NOTE]
> All data are stored in localStorage into browser.


> [!WARNING]
> No security, no admin right, no login.<br/>
> There may be some bugs.


## Features
- Players rankings after pools matches
- Playoff management
- Main bracket management
- Consolation bracket management
- Automatic players matching
- Anti-collision rule
- Interleaved placement
